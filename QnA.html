Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:
Gets or sets the HTML markup contained within an element.Can be manipulate to insert HTML content.
innerText:
Gets or sets the rendered text content of an element.Considers CSS styling and layout.
textContent:
Gets or sets the text content of an element and its descendants.Treats all content as plain text.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener can be attach to a parent element.

Question-4: What is event bubbling in the DOM?

When an event occurs on a DOM element, it does not just happen on that element. It propagates up the DOM tree, from the target element to its grandparent, and so on, until it reaches the root of the document.
